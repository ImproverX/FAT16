# FAT
Программа для работы с разделом FAT16 жёсткого диска ПК "Вектор-06ц" и "ПК-6128ц++"
Запускаются в любой версии МикроДОС, РДС, CP/M.

## Команды FAT
После запуска доступны команды:<br>
- **DIR [маска имени файла]** -- показать содержимое текущей директории
- **CD <директория>** -- смена директории
- **RUN <имя файла>.[COM|ROM|R0M] [параметры командной строки]** -- загрузить в память и запустить выполнение. В случае отсутствия расширения ищутся файлы в порядке: .COM, .ROM, .R0M. При запуске COM-файлов для них передаются параметры командной строки. Ограничение: загрузка и запуск возможен только файлов меньше 64кБ, проверка на свободное место в памяти не выполняется, т.е. возможно затирание ОС большим файлом.
- **CP [Диск]:<ИФ откуда> [Диск]:[ИФ куда]** -- копировать файл. Допускается использование масок, групповое копирование файлов. Ограничение: копируются только файлы меньше 48кБ, в текущей версии только с раздела FAT16 на другие диски ПК.
- **EXIT** -- выход в ОС.

## Драйвер FAT16.inc
Может открывать первый раздел жёсткого диска из MBR, читать каталоги, файлы. В текущей реализации может также записывать в файл с уменьшением/увеличением его размера, вносить изменения в таблицу FAT, но не может создавать новые файлы и директории, удалять и переименовывать файлы.

## Драйвер IDE.inc
Драйвер для работы с контроллером накопителя на жёстких магнитных дисках (НЖМД, он же HDD или SSD, CF-CARD...), подключённым по схеме ПК "Вектор-06ц" и "ПК6128ц++".
